Project Brief: Periodic Table Viewer

Overview

This project aims to develop an Angular application displaying a table of periodic elements with four columns: Number, Name, Weight, and Symbol. The application simulates downloading data on startup and provides editing capabilities. Additionally, a filter will allow users to search across all fields in the table.

Key Requirements

  • Table Display: Display a table with the columns: Number, Name, Weight, and Symbol. Use Angular Material’s table component for the layout.
  • Simulated Data Download: Simulate data loading during application startup and populate the table with predefined elements.
  • Data Editing Feature: Allow users to edit each row’s data through a popup dialog, and update the table after confirming changes.
  • Filter Functionality: Implement a search input to filter the table by all fields. Use a 2-second debounce delay for filtering.
